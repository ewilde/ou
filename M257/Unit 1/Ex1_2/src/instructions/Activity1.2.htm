Activity 1.2 - Developing a simple Robot class

Topic

This exercise allows you to test the simple Robot class discussed in the course text.

Materials

In addition to the Solution document, the project for this exercise contains two classes:

  • Robot, which is the complete class, as discussed in the course text;
  • TestRobot, which is incomplete.

Task

In the test class TestRobot, code to create three robot objects has been provided for you. You are to complete this class by adding statements to invoke some of the methods of the Robot class.

Instructions

Add statements to invoke various methods, and to display the new position of each robot, as indicated by the comments in the class TestRobot.

Build and run the program and check that the effect of these methods is as you would expect.
