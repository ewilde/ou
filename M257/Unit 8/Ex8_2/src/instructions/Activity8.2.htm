Activity 8.2 - Running the hare and the tortoise race again

Topic

In this activity we create threads by implementing the Runnable interface.

Materials

You will have to add your own classes to the project.

A discussion of the solution can be found in the Solution document.

Task

You will continue with the example of the "Hare and the Tortoise" race from Activity 8.1, now using a different approach for the threads. The classes developed in Activity 8.1 will need to be amended.

Instructions

Write new classes Hare and Tortoise based on the specification in Activity 8.1, but this time implementing the Runnable interface.

Write a new class ThreadRace2 which will create and start threads from the Hare and Tortoise classes. The class ThreadRace2 should behave like ThreadRace1 in all other respects.

Compile the program and run it several times. The output should be like that of Activity 8.1.
